# 🎬 What Should I Watch?

A beautiful pixel-art game that helps you decide what movie, TV series, or anime to watch based on your mood, available time, and preferences!

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.0+-green.svg)

## ✨ Features

- 🎨 Beautiful pixel-art nature theme with animated backgrounds
- 🌸 Swaying flowers, floating clouds, and flying butterflies
- 🔊 Sound effects for interactions
- 🎯 Smart recommendation system based on mood and time
- 📚 Extensive database of movies, series, and anime
- 🎮 Smooth 60 FPS gameplay

## 🚀 Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/what-should-i-watch.git
cd what-should-i-watch
```

2. Install dependencies:
```bash
pip install pygame
```

3. Run the game:
```bash
python watch_game.py
```

## 🎮 How to Play

1. Choose your content type (Movie, TV Series, or Anime)
2. Select your current mood
3. Pick how much time you have
4. Get a personalized recommendation!
